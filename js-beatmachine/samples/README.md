# Drum Samples for JS Beat Machine

This directory contains drum samples organized into kits:

- **CleanKit/**: kick01rr1.wav, snare01.wav, hhclosed.wav, hhopen.wav, clap.wav, tomlo.wav, congahi.wav, cowbell.wav
- **HotKit/**: Same sample names as CleanKit
- **WarmKit/**: Same sample names as CleanKit

## Where to Get Free Drum Samples

You can download free drum samples from the following resources:

1. **Freesound.org**: https://freesound.org/
   - A large collaborative database of creative-commons licensed sounds

2. **Bedroom Producers Blog**: https://bedroomproducersblog.com/free-samples/
   - Regularly updated with free sample packs

3. **Cymatics**: https://cymatics.fm/free-download-vault/
   - Free sample packs (account required)

4. **Splice**: https://splice.com/sounds
   - Some free samples available

5. **SampleFocus**: https://samplefocus.com/
   - Free drum samples and loops

## Sample Format

The samples in this project use WAV format. For best compatibility, ensure samples are:

- Short in duration (typically less than 2 seconds)
- Normalized to avoid volume inconsistencies
- Trimmed to remove excess silence

## Note

For licensing reasons, we don't include the drum samples directly in this repository. Please download your own samples and place them in this directory with the correct filenames.
